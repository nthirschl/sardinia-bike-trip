# Sardinia Trip

## Overview
A week-long trip to Sardinia, Italy in March 2026.

## Travelers
- **Noah** + **Wynne** (partners)
- **Fallon** (friend)

## Project Structure
- `itineraries/` — Arrival/departure times, daily schedules, and the master timeline for each traveler
- `accommodations/` — Lodging details: names, addresses, costs, booking status, check-in/out times
- `points-of-interest/` — Restaurants, cafes, sights, towns, and stops worth visiting

## Planned Outputs
1. **Visual Timeline** — A detailed, beautiful timeline showing when everyone arrives/departs, daily activities, locations, and accommodations.
2. **Interactive Map** — A map view of accommodations, transport, and points of interest.

## Conventions
- Dates use `YYYY-MM-DD` format.
- Currency in EUR unless noted.
- Coordinates stored as `lat, lng` decimal degrees.
- All data files use Markdown unless a specific format (GPX, GeoJSON) is more appropriate.
